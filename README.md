# PortPhoenix

 It is a portfolio builder web app that helps users create, customize, and preview professional portfolios in real-time.

---

## Features

* Template-based portfolio creation
* Live preview while editing
* Basic customization (colors, layout)
* Fetch templates via API

---

## Tech Stack

* React
* CSS / Tailwind

## 🔌 API Endpoints

| Method | Endpoint       | Description         |
| ------ | -------------- | ------------------- |
| GET    | /templates     | Fetch templates     |
| POST   | /portfolio     | Save portfolio data |
| GET    | /portfolio/:id | Get saved portfolio |
